# Hello rails back end:

A Rails project as API.

## Built With

- Ruby
- Rails
- PostgreSQL

## Getting Started

To get a local copy up and running follow these simple example steps.

## Instructions

```
$ cd <folder>
```

```
$ git clone git@github.com:PolinaStamenova/hello-rails-back-end.git
```

```
$ cd hello-rails-back-end
```

## Install

```
$ bundle install
```

## Run

```
$ rails db:create
```

```
$ rails db:migrate
```

```
$ rails db:seed
```

```
rails s
```
